# 📌 프로젝트 기획

## 1. 프로젝트 개요

출근 시간과 퇴근 시간 기준으로 옷차림을 추천해주는 날씨 앱

---

## 2. 문제 정의

* 아침마다 옷 선택이 번거롭다
* 일교차로 인해 옷 선택 실패 경험
* 미세먼지까지 고려해야 하는 불편함

👉 기존 날씨 앱은 “정보”만 제공하고 “결정”은 도와주지 않는다

---

## 3. 해결 방법

* 날씨 + 미세먼지 + 시간대 데이터를 종합
* 사용자에게 **즉시 실행 가능한 옷차림 추천 제공**

---

## 4. 주요 기능

* 현재 위치 기반 날씨
* 옷차림 추천
* 출근/퇴근 시간 날씨
* 7일 예보
* 미세먼지 정보

---

## 5. 타겟 사용자

* 출근하는 직장인
* 아침에 빠르게 의사결정이 필요한 사용자

---

## 6. 차별화 포인트

* 단순 날씨 제공 ❌
* 행동(옷 선택)까지 연결 ⭕

---

## 7. 수익 모델

* 옷 추천 → 제휴 링크
* 마스크 추천 → 구매 링크
* 광고 (AdMob)

---

## 8. 향후 확장

* 개인 체감 온도 학습
* 코디 추천
* 알림 기능
