# 🔥 트러블슈팅

## 1. 위치 정보 가져오기 실패

### 문제

로컬 환경에서 위치 정보가 정상적으로 가져와지지 않음

### 원인

* 브라우저 권한 제한
* Expo 환경 미사용

### 해결

* Expo Go 앱에서 실행
* 권한 요청 코드 추가

---

## 2. API 호출 실패

### 문제

날씨 데이터가 불러와지지 않음

### 원인

* API 키 오류
* 네트워크 문제

### 해결

* API 키 확인
* try-catch 적용
* fallback UI 추가

---

## 3. 데이터 undefined 오류

### 문제

데이터 접근 시 에러 발생

### 원인

* 비동기 처리 문제

### 해결

* optional chaining 사용
* 로딩 상태 분리

## 4. UX 문제

사용자가 정보를 한눈에 이해하기 어려움

### 해결

* 카드 UI 적용
* 정보 우선순위 재배치
