# 📱 오늘의 옷차림 (Weather & Outfit)

> 오늘 뭐 입지?
> 오늘의 최저/최고 기온을 기준으로 **옷차림을 추천해주는 날씨 앱**

---

## 🔥 프로젝트 소개

아침마다 “오늘 뭐 입지?” 고민하는 문제를 해결하기 위해 만든 앱입니다.

단순한 날씨 정보 제공이 아니라
👉 **일교차를 고려한 오늘의 옷차림을 추천**하는 것이 핵심입니다.

---

## 🎯 기획 의도

* 아침마다 옷 선택이 번거롭다
* 일교차 때문에 옷 선택이 어렵다
* 미세먼지까지 고려해야 한다

👉 이 문제를 한 번에 해결하는 앱을 만들고자 했습니다.

---

## ✨ 주요 기능

### 📍 현재 위치 기반 날씨

* 위치 자동 감지
* 현재 온도 / 체감 온도 / 바람

### 👕 옷차림 추천

* 온도 + 바람 + 미세먼지 기반 추천
* 한국 날씨 기준 현실적인 로직 적용

### 📅 일교차 고려

* 시간대별 날씨 제공
* 각각 다른 옷차림 추천

### 🌫️ 미세먼지 정보

* 공기질 상태 표시
* 마스크 착용 여부 안내

### 📊 5일 예보

* 날짜별 날씨 정보
* 옷차림 추천 포함
